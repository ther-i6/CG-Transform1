# 计算机图形学实验报告（精简版）
## 一、实验名称
基于 Taichi 的 MVP 坐标变换实验

## 二、实验目的
1. 理解并掌握**模型-视图-投影（MVP）**变换的原理与流程。
2. 手动实现**模型旋转、视图变换、透视投影**三个核心矩阵。
3. 使用 Taichi 完成 3D 点到 2D 屏幕的投影与渲染，实现可旋转三角形。

## 三、实验原理
1. **模型变换**：绕 Z 轴旋转，将物体从局部坐标转到世界坐标。
2. **视图变换**：将相机平移至原点，构建观察坐标系。
3. **投影变换**：通过透视投影将视锥体映射到标准设备坐标（NDC）。
4. **透视除法**：齐次坐标除以 w 分量，得到归一化坐标并映射到屏幕。

## 四、实验环境
+ 语言：Python
+ 框架：Taichi
+ 功能：MVP 矩阵实现、3D 三角形渲染、键盘交互旋转

## 五、实验内容
1. 定义 3D 三角形顶点并使用齐次坐标表示。
2. 实现绕 Z 轴的模型矩阵、相机视图矩阵、透视投影矩阵。
3. 对顶点进行 MVP 变换、透视除法与屏幕映射。
4. 绘制**无填充、彩色粗边框**三角形，支持 A/D 键旋转。

## 六、实验结果
+ 成功显示 3D 透视效果的三角形。
+ 可通过键盘控制旋转，画面清晰、运行稳定。
+ MVP 变换正确，透视投影符合近大远小规律。
+ 

## 七、实验总结
本次实验完成了从 3D 坐标到 2D 屏幕的完整变换流程，深入理解了 MVP 矩阵的作用与图形渲染管线，掌握了 Taichi 基础渲染与向量、矩阵操作。
